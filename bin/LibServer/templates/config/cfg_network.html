Network configuration

Here you will find configuration options for the network adapters that {{config.product}} is set up to configure. To add one not listed here, simply add to your local configuration file the following lines:

[interface.device name]
ui_label=a short name
type=wired or wireless
enabled=true
; there are two modes for wireless adapters: Client and AP. In client mode, the network is
; connected to via the /etc/network/interfacces mechanism. Otherwise, hostapd is used.
mode=ap or client
dhcp=true or false
ip=ip address
subnet_mask=subnet mask
gateway=gateway, or nothing
; wireless options
security=wpa or none
psk=allonewordlowercase
ssid=cheesecake
